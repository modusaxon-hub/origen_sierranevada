# 📁 Scripts — Desarrollo & Despliegue

Todos los scripts de desarrollo y despliegue están organizados en la carpeta **`scripts/`**

## 🚀 Inicio Rápido

```bash
# Primer setup
scripts/01_instalar_dependencias.bat

# Desarrollo diario
scripts/02_iniciar_desarrollo.bat
```

## 📋 Lista Completa

| # | Script | Propósito |
|---|--------|-----------|
| 1️⃣ | `01_instalar_dependencias.bat` | Instala node_modules (una sola vez) |
| 2️⃣ | `02_iniciar_desarrollo.bat` | Inicia dev server en puerto 5000 |
| 3️⃣ | `03_validar_proyecto.bat` | Verifica errores TypeScript |
| 4️⃣ | `04_compilar_produccion.bat` | Genera build optimizado |
| 5️⃣ | `05_copiar_a_htdocs.bat` | Copia a staging (InfinityFree) |
| 6️⃣ | `06_limpiar_proyecto.bat` | Limpia node_modules + build |
