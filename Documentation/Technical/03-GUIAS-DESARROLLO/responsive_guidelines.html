Protocolo Mobile-First

📏 Breakpoints Estándar

  • Móvil (xs): < 640px
  • Tablet (md): 768px
  • Laptop (lg): 1024px
  • Desktop (xl): 1280px

🛠️ Reglas de Comportamiento

1. Tipografía Adaptativa:

Uso de escalas progresivas. Ejemplo: text-3xl md:text-5xl lg:text-6xl. El título nunca debe desbordar horizontalmente.

2. Touch Targets:

Botones con altura mínima de 44px (h-12) para garantizar accesibilidad táctil fluida.

3. Inputs de Formulario:

Mínimo 16px (text-base) para evitar el zoom automático intrusivo de iOS.
