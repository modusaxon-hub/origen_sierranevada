Guía de Acceso Remoto

🌐 Protocolo: Zrok Tunnel + Vite Server

El sistema utiliza un túnel de reenvío seguro para permitir pruebas externas antes del despliegue final. 

URL Pública: https://origen2025.share.zrok.io
Host Local: 127.0.0.1:3000

Configuración del Servidor

Vite debe permitir conexiones externas mediante allowedHosts para evitar errores 500.

// vite.config.ts
export default defineConfig({
  server: {
    port: 3000,
    host: '127.0.0.1',
    allowedHosts: true
  }
});

🚨 Manual de Contingencia

Error Solución Técnica
502 Bad Gateway Asegúrese de que el servidor local (NPM) esté activo.
Port in use Libere el puerto 3000 con taskkill en PowerShell.
Timeout Reinicie el agente zrok local.
