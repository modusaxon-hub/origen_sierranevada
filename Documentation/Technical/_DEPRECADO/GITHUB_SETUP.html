Origen Sierra Nevada - Deployment

Guía de Carga a GitHub

Paso 1: Crear del Repositorio

Ve a github.com/new y crea un repositorio privado llamado origen-sierra-nevada.

⚠️ Nota: No inicialices con README, .gitignore o licencia, ya que el proyecto local ya los tiene.

Paso 2: Conexión Local

Ejecuta estos comandos en tu terminal de PowerShell:

git remote add origin https://github.com/manuelpertuz624-eng/origen-sierra-nevada.git
git branch -M main
git push -u origin main

Paso 3: Seguridad de Credenciales

✅ Seguridad Garantizada: Tu archivo .env está en el .gitignore, por lo que nunca se subirá a la nube pública.
