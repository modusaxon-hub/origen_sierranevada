Origen Sierra Nevada - UI/UX Guidelines

Jerarquía Responsiva

📱 Mobile (< 1024px)

Prioridad vertical descendente. Los elementos se apilan para garantizar legibilidad en pantallas estrechas (Pixel, iPhone, Honor X8a).

  • Selector → Título → Imagen → Botón Compra.
  • Gestos táctiles habilitados (Swipe).
🖥️ Desktop (> 1024px)

Composición de "Exhibidor Maestro" aprovechando el ancho de banda visual. El contenido fluye hacia los paneles laterales liberando el centro para el arte del producto.

Navegación de Extremos

Las flechas de navegación están ancladas a los bordes absolutos del viewport para no interrumpir la composición visual central.
