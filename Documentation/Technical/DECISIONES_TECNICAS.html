⚠️ DECISIÓN CRÍTICA: IVA Incluido en Precio

Comercialmente, Origen Sierra Nevada maneja precios "todo incluido". El sistema debe extraer la base gravable del total para efectos contables DIAN sin alterar el cobro final al cliente.

const IVA_RATE = 0.19;
const baseGravable = Math.round(precioExhibido / (1 + IVA_RATE));
const ivaIncluido = precioExhibido - baseGravable;
