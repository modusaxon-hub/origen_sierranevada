👥 Casos de Uso Quirúrgicos

UC-ADMIN-01: Aprobación del Círculo Interno

Actor: Administrador

Tensión: La exclusividad de la marca depende de este filtro manual. Si falla, el Brandbook queda expuesto a terceros.

1. Localización de usuario en "Pendientes" → 2. Validación de mérito → 3. Concesión de acceso → 4. Activación de Welcome Email.

UC-ADMIN-02: Configuración de Esencia (Intrinsics)

Actor: Curador de Marca

Tensión: Sin arquetipos definidos, el producto pierde su valor narrativo diferencial.

1. Elección de café → 2. Asignación de Arquetipo/Mood → 3. Publicación instantánea en Despensa.

UC-CLIENT-03: Validación de Pago Manual

Actor: Cliente

Tensión: La confianza en la transferencia depende de la capacidad de reportar el pago rápidamente.

1. Finalización de pedido → 2. Captura de comprobante → 3. Subida a Supabase Storage → 4. Notificación automática a Admin.
